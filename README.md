## Projetos

- **Mapa Eleições Fabrício Rosa** — Mapas coropléticos interativos com resultados das eleições proporcionais em Goiás (2016–2024).
- **Painel Vereadores Goiás** — Dashboard com série histórica de candidatos a vereador em todos os municípios de Goiás (2004–2024).
